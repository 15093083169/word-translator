划词翻译 — 选中文字，双击 Ctrl

📖 使用方法

1. 在网页上选中要翻译的文字

2. 快速按两下 Ctrl 键

3. 翻译结果会以浮动窗口显示

4. 点击「复制翻译」按钮复制结果

💡 获取 API Key：open.bigmodel.cn
